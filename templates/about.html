À propos de Dark Gpt Ai

🎯 Description

Dark Gpt Ai est un assistant IA sans censure basé sur Dolphin 2.9.1 Mistral 7B, spécialement optimisé pour tourner sur des machines avec 8GB de RAM grâce à la quantification 4-bit.

⚙️ Technologies

  • Modèle: Dolphin 2.9.1 Mistral 7B (4-bit)
  • Framework: Hugging Face Transformers
  • Backend: Flask + Gunicorn
  • Frontend: HTML5, CSS3, JavaScript
  • Quantification: bitsandbytes (NF4)

📊 Caractéristiques

  • ✅ Sans censure - Aucune restriction
  • ✅ 7B paramètres - Modèle puissant
  • ✅ 4-bit quantifié - Seulement 4GB RAM
  • ✅ Support multilingue - Français, Anglais, etc.
  • ✅ Expert en code - Python, JavaScript, etc.
  • ✅ API REST - Intégration facile
