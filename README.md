# 🔥 Dark Gpt Ai

Assistant IA sans censure basé sur **Dolphin 2.9.1 Mistral 7B**, optimisé pour **8GB RAM** avec quantification 4-bit.

## ✨ Caractéristiques

- ✅ **Sans censure** - Réponses complètes et sans restrictions
- ✅ **7B paramètres** en 4-bit (seulement ~4GB RAM)
- ✅ **Excellent en code** - Python, JavaScript, etc.
- ✅ **Interface web moderne** avec chat en temps réel
- ✅ **API REST** complète
- ✅ **Optimisé CPU** - Tourne parfaitement sur 8GB RAM
- ✅ **Service systemd** - Démarrage automatique
